# corefunds-performance-benchmarking
Comparative analysis of core thematic funds vs. global benchmarks. Identifying structural alpha and performance decay during regime shifts.

CoreFunds Performance Benchmarking: The "Narrative vs. Structure" Audit
This repository provides a quantitative framework to evaluate the performance of "Core" thematic mutual funds against global benchmarks (e.g., VTI, VT).

The goal is to determine if active management is delivering Structural Alpha (skill) or simply Tempered Beta (market exposure) layered with high management fees and narrative-driven marketing.

Methodology
Analyze funds using four primary quantitative filters to strip away "Closet Indexing":

Rolling Beta Analysis: Identifies the fund's true correlation to equity markets. Most "Core" funds cluster between 0.6 and 0.9, suggesting they are a leveraged version of the index rather than a different asset class.

Rolling Information Ratio (IR): Measures the consistency of the manager's excess return relative to the risk (tracking error) they take.

Alpha Z-Score: Normalizes the manager's alpha to identify if outperformance is a persistent signal or a mean-reverting outlier.

Probability of Luck (P-Value): Calculates the statistical probability that the fund’s outperformance was generated by random chance (H0: No Skill).
